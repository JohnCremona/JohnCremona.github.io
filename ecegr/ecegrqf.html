Elliptic Curves with Everywhere Good Reduction over Quadratic Fields
compiled by
J. E. Cremona
University of Warwick, U.K.

Updated 2005-09-29 (with minor correction 2017-02-21)

Note that this table is no longer being updated, as it has been superceded by the LMFDB, which as of February 2017 contains 497 elliptic curves with everywhere good reduction.

We give tables showing the existence or non-existence of elliptic curves with everywhere good reduction defined over quadratic number fields Q(√d). This data is currently incomplete and will be updated. Where possible we give attributions and references: please email me if you see any errors or omissions.

Where we list no curves, we distinguish between cases where none are known but it is not proved that none exist from cases where it has been proved that non exist. Similarly when some curves are known to exist, we indicate whether or not the curves listed are known to be a complete list.

There are separate tables for real and imaginary quadratic fields. At present the tables give data for real fields Q(√d) for 1<d<100 and data for imaginary fields Q(√d) with 0>d>-100. For curve details (where available) follow the link on the value of d.
