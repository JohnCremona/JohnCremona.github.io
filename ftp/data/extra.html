  1. Systematic coverage of all levels:
    • all data up to level 350000 are available from the main page.

  2. Tables of elliptic curves with good reduction outside certain sets S of primes. These were found using an implementationof the algorithm given in Cremona and Lingham "Finding all elliptic curves with good reduction outside a given set of primes", Experimental Mathematics 16 No.3 (2007), 303-312, which is completely independent of modular computations. They are sorted by conductor, isogeny class and curve and labelled using LMFDB labels.
    • S={2,3}: (752 curves), all in main database
    • S={2,3,5}: (7600 curves) (corrected 2013-10-13, there were 48 missing)
    • S={2,3,7}: (7168 curves)
    • S={2,3,5,7}: (71520 curves) (corrected 2013-10-23, there were 192 missing)
    • S={2,3,11}: (6640 curves)
    • S={2,13}: (336 curves), all in main database
    • S={2,17}: (256 curves), all in main database
    • S={2,19}: (336 curves), all in main database
    • S={2,23}: (256 curves), all in main database
