This table contains an entry for the strong Weil curve in each isogeny
class
\footnote{This is the first curve in the class except for class 990H,
where we give a generator for the curve 990H3.}  of positive rank.
For each we give the $(x,y)$ coordinates of generators of the
Mordell--Weil group (modulo torsion) with respect to the minimal
equation of Table~1. In a few cases the coordinates are not integral,
in which case we give them in the form $(a/c^2, b/c^3)$ with
$a,b,c\in\Z$ and $\gcd(a,b,c)=1$.
