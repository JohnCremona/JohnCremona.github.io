This table is largely self-explanatory.  There is one row for each 
rational newform $f$ for $\G0(N)$ for $N\le1000$; forms at the same level
are given an identifying letter as in Table~1, together with the Antwerp 
code for $N\le200$.  The other columns contain the Hecke eigenvalues of 
$f$ for primes up to 100: either $T_p$, when $p\ndiv N$, or $W_q$ when 
$q\div N$.  The latter are indicated in the table simply as $+$ or $-$ to 
distinguish them.   Where the largest prime divisor $q$ of $N$ is greater 
than $100$, the extra value $\eps_q$ is entered in the right-most column: 
there is only ever at most one such prime $q>100$.
